## WaifUwU

An AI girlfriend that talks and speaks to you.
She listens to you always & will never leave you !!! 😍
Why chase real life women when you can have this computer-Waifu ???

## Main Features

1. A beautiful 3D model with animations
2. Generative AI integration for speech-synthesis & text responses
3. Manual chat context caching at the server side so your Waifu responds according to previous conversations

### To run/develop locally,

1. Install pnpm on machine (recommended)
2. Goto "node/"
3. Add your API keys and MONGO_URI in a .env file (refer to .envSample) and run `pnpm i && pnpm run dev`
4. Goto "react/"
5. Run `pnpm i && pnpm run dev` and you should see vite give a port: http://localhost:3000 or similar.
6. Go there and enjoy !

### APIs used as of writing:

GEMINI_API for AI-bot-responses,
SPEECHIFY_API for AI-speech-synthesis

### Major Tech Stack:

React, Express, Mongoose (MongoDB), Three.js : React-three-fiber
